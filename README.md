# Витрины по данным из [Теста TPC-H](https://www.tpc.org/tpch/)
## Перечень витрин (сводная информация)
+ позиции каждого заказа
+ заказы в разрезе месяца, страны отправки и приоритета
+ данные по клиентам в разрезе региона и страны заказа, маркетингового сегмента
+ показатели по поставщикам в разрезе региона и страны заказа
+ информация по грузоперевозкам в разрезе страны и типа поставки, типа контейнера
## Особенности расчета
+ чтение и итоговая запись витрин в рамках S3 хранилища
+ идемпотентность ETL-процедур
+ параллельное выполнение ETL-процедуры для каждой витрины
## Технические фишки
+ использование *Airflow* в качестве ETL-оркестратора
+ использование *PySpark* на *Kubernetes*
+ применение *Greenplum PXF* (внешние таблицы Greenplum над данными из S3)
